# 💖 Love Quiz

A fun and interactive relationship quiz built using HTML, CSS and JavaScript.

This project is designed as a playful prank website where users must make a difficult choice between YES and NO. The experience includes moving buttons, funny reactions, custom images, multiple screens and surprise endings.

---

## ✨ Features

- 🎯 Interactive multi-step love quiz
- 🏃 Moving button mechanics
- 🐱 Custom meme image support
- ❤️ Romantic themed UI
- 📱 Mobile responsive design
- 🎭 Funny heartbreak screens
- 🎵 Background music support
- 🚀 Hosted using GitHub Pages

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🎮 How It Works

1. User is asked a simple question.
2. The YES button keeps running away.
3. Choosing NO unlocks funny reactions.
4. Multiple screens create a prank-like experience.
5. The journey ends with a final surprise message.
